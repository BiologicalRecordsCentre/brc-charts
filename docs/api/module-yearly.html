metrics Array.<Object> An array of objects, each describing a numeric property in the input data for which graphics should be generated on the chart. Each of the objects in the data array can be sepecified with the properties shown below. (The order is not important.)
  • prop - the name of the numeric property in the data (count properties - 'c1' or 'c2' in the example below).
  • label - a label for this metric. (Optional - the default label will be the property name.)
  • colour - optional colour to give the graphic for this metric. Any accepted way of specifying web colours can be used. Use the special term 'fading' to successively fading shades of grey. (Optional - default is 'blue'.)
  • opacity - optional opacity to give the graphic for this metric. (Optional - default is 0.5.)
  • linewidth - optional width of line for line for this metric if displayed as a line graph. (Optional - default is 1.)
data Array.<Object> Specifies an array of data objects. Each of the objects in the data array must be sepecified with the properties shown below. (The order is not important.)
  • taxon - name of a taxon.
  • year - a four digit number indicating a year.
  • c1 - a count for a given time year (can have any name).
  • c2 - a count for a given time year (can have any name). ... - there must be at leas one count column, but there can be any number of them.
taxa Array.<string> An array of taxa (names), indicating which taxa create charts for. If empty, graphs for all taxa are created. (Default - [].)
minYear number Indicates the earliest year to use on the y axis. If left unset, the earliest year in the dataset is used. (Default - null.)
maxYear number Indicates the latest year to use on the y axis. If left unset, the latest year in the dataset is used. (Default - null.)
