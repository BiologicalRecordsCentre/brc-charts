metrics Array.<Object> An array of objects, each describing a numeric property in the input data for which a line should be generated on the chart. Each of the objects in the data array must be sepecified with the properties shown below. (The order is not important.)
  • prop - the name of the numeric property in the data (count properties - 'c1' or 'c2' in the example below).
  • label - a label for this metric.
  • colour - optional colour to give the line for this metric. Any accepted way of specifying web colours can be used. Use the special term 'fading' to successively fading shades of grey.
  • fill - optional colour to colour the graph area for this metric. Any accepted way of specifying web colours can be used.

data Array.<Object> Specifies an array of data objects. Each of the objects in the data array must be sepecified with the properties shown below. (The order is not important.)
  • taxon - name of a taxon.
  • week - a number between 1 and 53 indicating the week of the year.
  • c1 - a count for a given time period (can have any name).
  • c2 - a count for a given time period (can have any name). ... - there must be at leas one count column, but there can be any number of them.
