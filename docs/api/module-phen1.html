  • label - a label for this metric.
  • colour - optional colour to give the line for this metric.
data Array.<Object> Specifies an array of data objects. Each of the objects in the data array must be sepecified with the properties shown below. (The order is not important.)
  • taxon - name of a taxon.
  • week - a number between 1 and 53 indicating the week of the year.
  • p1 - a count for the first time period.
  • p2 - a count for the second time period.
