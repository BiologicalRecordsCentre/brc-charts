data Array.<Object> Specifies an array of data objects. Each of the objects in the data array must be sepecified with the properties shown below. (The order is not important.) For each taxon, there is assumed to be a single data item for each rounded distance/altitude combination for which there is a metric value.
  • taxon - The taxon for which this point refers.
  • distance - The distance north (OS northing) in kilometers rounded down to the nearest 50 km.
  • altitude - The altitude (asl) in metres rounded down to the nearest 100 m.
  • metric - a metric (number) for the given distance and altitude.
ranges Array.<Object> Specifies an array of objects defining ranges for displaying the metrics. Each of the objects in the data array must be sepecified with the properties shown below. (The order affects how they appear in the legend.)
  • min - The minimum metric value to be included in this range.
  • max - The maximum metric value to be included in this range.
  • radius - The radius of the dot to be drawn for this range.
  • legend - Text for the legend item for this range.
taxa Array.<string> An array of taxa (names), indicating which taxa create charts for. If empty, graphs for all taxa are created. (Default - [].)
