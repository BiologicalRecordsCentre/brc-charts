  • name - the name of the data item uniquely identifies it and is shown in the legend.
  • number - a numeric value associated with the item.
  • colour - an optional colour for the symbol which can be hex format, e.g. #FFA500, RGB format, e.g. rgb(100, 255, 0) or a named colour, e.g. red. If not specified, a colour will be assigned.
  • image - this optional property allows you to specify the url of an image file which can be displayed when a user selects the associated item.
